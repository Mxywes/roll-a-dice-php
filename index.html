Roll a dice! - A PHP app

"Roll a dice" is just a simple nearly one-file PHP script that enables the user to roll a dice, with 4, 6, 8 or 9 sides.


  • Learn more

Find source on GitHub

The app is hosted on a GitHub repository, from where you can download it and deploy to a server with working PHP support.

The Random Compatibility Library is included to enable PHP 5.x support of random_int(). You don't need it if you're using PHP 7 or earlier.

Deployed on Heroku

A deployed, up-to-date with the current GitHub master branch, working version of the app is deployed to Heroku for testing purposes or simply using the app without having to upload it to a server yourself.
